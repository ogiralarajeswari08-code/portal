# Car Portal - Backend

Minimal Express + Mongoose backend for the Car Portal project.

Features:
- Auth (register/login) with JWT
- Car record CRUD endpoints
- File uploads (photos, video) saved to `uploads/` and served statically

.env.example provided. Run:

1. npm install
2. copy .env.example to .env and edit
3. npm run dev

API endpoints are mounted under `/api`.
